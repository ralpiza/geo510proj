# Installation

## Stable release

To install geo510proj, run this command in your terminal:

```
pip install geo510proj
```

This is the preferred method to install geo510proj, as it will always install the most recent stable release.

If you don't have [pip](https://pip.pypa.io) installed, this [Python installation guide](http://docs.python-guide.org/en/latest/starting/installation/) can guide you through the process.

## From sources

To install geo510proj from sources, run this command in your terminal:

```
pip install git+https://github.com/ralpiza/geo510proj
```
